Quick Start

1 — Before your LLM call

import httpx

slice = httpx.post(
    "http://localhost:5000/slice",
    headers={"X-API-Key": "rudi_sk_..."},
    json={"task": user_message},
).json()

# Use slice["system"] as system prompt
# Use slice["prompt"] as user message
# Pass slice["inject_ids"] to /decisions below

2 — After your LLM call

    "http://localhost:5000/decisions",
    headers={"X-API-Key": "rudi_sk_..."},

    },
